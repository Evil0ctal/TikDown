### TikDown
This is a Tiktok/Douyin video downloader, build by Electron. To download video without watermark by only pasting shared link.
这是一个使用了Electron构建的 Tiktok/抖音 下载器。通过粘帖剪贴板里的分享链接下载无水印的视频。
